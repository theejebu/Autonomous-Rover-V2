# Autonomous Rover V2
This is a WIP version 2 of my [autonomous face tracking rover](https://github.com/theejebu/Autonomous-Face-Tracking-Rover). This is not a tutorial, but information on how I made it. 

### Components
- 4 mecanum wheels
- 4 DC motors
- L9110S 4 channel motor drive module
- A chassis for the components to lie on
- 18650 battery (mine was rechargeable)
- ESP32-S3-CAM
- Some extra jumper wires

### Construction
To make the autonomous rover, I first dissasembled a mecanum based RC car, which provided many of the key components provided above. (Image below)
